# Project-Retail-Analytics
 Retail Analytics: Predicting customer purchase behaviour
 The goal is to analyze customer purchase behavior using retail transaction data. Explore customers' demographic information, purchasing patterns, and interactions with marketing campaigns to identify key insights. The focus is on understanding customer preferences and optimizing promotional strategies to maximize marketing effectiveness and return on investment (ROI).This project involves analyzing transaction data to uncover patterns that help improve business decisions and customer.
Understand customer purchase patterns using transaction data. Data exploration, identifying top products, customer segmentation by age groups, and purchase frequency.
